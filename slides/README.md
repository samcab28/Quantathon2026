# slides/

Presentación de **5 minutos**, entregable obligatorio del reto.

## Qué debe contener

- Archivo fuente de las diapositivas (PowerPoint, Google Slides exportado,
  Markdown+Marp, LaTeX Beamer, etc.) y su export a PDF/PPTX final.
- Contenido sugerido, alineado con la rúbrica de evaluación:
  1. Problema y motivación (ODS 6, impacto real del acceso a agua potable).
  2. Línea base clásica: metodología y resultados.
  3. Kernel cuántico: mapa de características, diagrama de circuito, heatmap.
  4. Comparación clásico vs. cuántico (métricas + barras de error) y
     resultado de escalado.
  5. Limitaciones honestas.
  6. (Opcional) resultados de hardware real / mitigación de ruido.
- Debe poder explicarse técnicamente sin apoyo (criterio "Explicación", 20%
  de la rúbrica) — evitar diapositivas que solo el informe explique.

Usar las mismas cifras y figuras de `results/` que el informe técnico, para
que ambos entregables sean consistentes.
